# Object tracking HSV calibrator using openCV

An interactive tool designed to tune the HSV ranges for object detection and tracking

## How it works
- Trackbars are used to adjust HSV values in real time
- Mask output is displayed
- Optimal thresholds for object detection are determined

## Run
- pip install -r requirements.txt
- python main.py
- Use trackbars to adjust HSV ranges and observe the mask output in real time.
